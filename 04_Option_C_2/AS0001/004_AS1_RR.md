#### 使能MPLS LDP能力

```text
router ospf 1
 mpls ldp autoconfig

```

#### 配置BGP IPv4，通告本地loopback0路由，并分别标签

```text
router bgp 1
 no bgp default ipv4-unicast
 neighbor 10.0.1.1 remote-as 1
 neighbor 10.0.1.1 update-source Loopback0
 neighbor 10.0.1.3 remote-as 1
 neighbor 10.0.1.3 update-source Loopback0
 !
 address-family ipv4
  network 10.0.1.254 mask 255.255.255.255
  neighbor 10.0.1.1 activate
  neighbor 10.0.1.1 route-reflector-client
  neighbor 10.0.1.1 send-label
  neighbor 10.0.1.3 activate
  neighbor 10.0.1.3 route-reflector-client
  neighbor 10.0.1.3 send-label

```

#### 配置MP-BGP

```text
router bgp 1
 no bgp default ipv4-unicast
 neighbor 10.0.1.1 remote-as 1
 neighbor 10.0.1.1 update-source Loopback0
 neighbor 10.0.2.254 remote-as 2
 neighbor 10.0.2.254 update-source Loopback0
 neighbor 10.0.2.254 ebgp-multihop 255
 
 !
 address-family vpnv4
  neighbor 10.0.1.1 activate
  neighbor 10.0.1.1 send-community extended
  neighbor 10.0.1.1 route-reflector-client
  neighbor 10.0.2.254 activate
  neighbor 10.0.2.254 send-community extended
  neighbor 10.0.2.254 next-hop-unchanged

```
