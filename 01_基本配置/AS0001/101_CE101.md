#### 初始化
```text
#
hostname CE101
no ip domain lookup
interface Loopback0
 ip address 10.1.1.1 255.255.255.255
!
interface Ethernet0/0
 ip ospf network point-to-point
 ip address 172.16.1.254 255.255.255.0
 no shutdown
!
line con 0
 exec-timeout 0 0
 logging synchronous
#
#### 配置OSPF
#
router ospf 1001
 prefix-suppression
 network 0.0.0.0 255.255.255.255 area 0
#
```
