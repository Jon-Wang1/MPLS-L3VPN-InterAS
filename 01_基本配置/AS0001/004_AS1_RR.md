#### 初始化
```text
#
hostname AS1_RR
no ip domain lookup
interface Loopback0
 ip address 10.0.1.254 255.255.255.255
!
interface Ethernet0/0
 ip address 192.168.1.1 255.255.255.0
 no shutdown
!

line con 0
 exec-timeout 0 0
 logging synchronous
#
```

#### 配置IGP（OSPF）
```text
#
router ospf 1
network 10.0.1.254 0.0.0.0 area 0
network 192.168.1.1 0.0.0.0 area 0
#
```
